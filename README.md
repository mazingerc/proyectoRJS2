# NBA Caps Import

Proyecto de ecommerce para importación de gorras de la NBA.

## Descripción

Este proyecto es un ecommerce desarrollado en React y Vite que te permite explorar y comprar gorras de la NBA importadas directamente desde los Estados Unidos. Ofrecemos una amplia selección de gorras de tus equipos favoritos. ¡Explora nuestro catálogo y encuentra la gorra perfecta para ti!

## Características

- Navega por una variedad de gorras de la NBA.
- Filtra por conferencia u oferta.
- Agrega gorras a tu carrito de compras.
- Finaliza la compra con nuestro sencillo proceso de pago.
- Interfaz de usuario atractiva y fácil de usar.

## Instalación

1. Clona este repositorio:
   ```bash
   git clone https://github.com/mazingerc/proyectoRJS2

2. Navega a la carpeta del proyecto:
cd nba-caps-ecommerce

3. Instala las dependencias:
npm install

4. Inicia la aplicación:
npm run dev

5. Abre tu navegador y accede a http://localhost:3000 

## Uso

Explora nuestro catálogo de gorras de la NBA y encuentra la tuya.
Utiliza las opciones de filtrado para personalizar tu búsqueda.
Agrega tus gorras favoritas al carrito de compras.
Finaliza tu compra de manera sencilla.
